A propos

Projet Réseau S2 4BIM INSA Lyon, Louise Gsell, Hanâ Lbath, Emilie Mathian

Objectif de ce site web

Il permet d’analyser des séquences protéiques ou nucléiques (ADN) au format fasta. En ce qui concerne l'analyse de séquences nucléiques, ce site permet de connaître la composition de la séquence, le pourcentage de C+G et le nombre d'îlot CpG. En ce qui concerne les séquences protéiques, on peut calculer le nombre de résidus hydrophobes, le pourcentage de résidus chargés et la charge nette de la séquence.

Fonctionnement de ce site web

Sur la page d'accueil, l'utilisateur peut choisir s'il souhaite effectuer une analyse de séquence protéique ou nucléique. Il peut aussi choisir s'il souhaite effectuer l'analyse d'une séquence se trouvant sur les bases de données en ligne Uniprot ou EMBL. Pour ce faire, l'utilisateur rentre l'identifiant de la séquence dans le champ dédié. L'utilisateur peut aussi choisir d'anaylyser plutôt une séquence qu'il a déjà au format fasta. Pour ce faire, il doit copier sa séquence (au format fasta) dans le champ dédié. L'utilisateur peut alors lancer l'analyse en cliquant sur le bouton "submit".

Lors de son fonctionnement, le site récupère la séquence fasta souhaitée et crée des fichiers résultats tabulés (fichiers « .txt »). Ces données sont créées de manière automatique et sont classées dans un dossier portant comme nom « Analyse_ » suivit de la description de la séquence étudiée. Les résultats sont directement affichés dans une page dédiée à cette analyse. Des graphiques (fichiers « .png ») se basant sur l'analyse sont également créés dans ce dossier (à condition que la machine soit équipée du module matplotlib, dans le cas contraire il ne crée que les fichiers tabulés). Les graphes sont affichés de même sur la page de résultats d'analyse.

Dans la barre de recherche, l'utilisateur peut effectuer des recherches exactes sur les séquences dont l'analyse a déjà été effectuée. Il peut également effectuer des recherches d'expressions régulières en écrivant sa requête de la façon suivante : .*requete.*
